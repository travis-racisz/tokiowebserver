## Tokio Web Server

### A Rust web server to serve templated HTMX
#### Utilizing 
- Tokio - Async Rust
- Axum - Routing
- Askama - Templating HTML
- HTMX - Frontend funcitionality in HTML such as data fetching and replacing elements

### Installation
```bash
  git clone https://github.com/travis-racisz/tokiowebserver
  cd tokiowebserver
  cargo run
```
